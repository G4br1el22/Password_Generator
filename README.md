# Password Generator (v1)

A simple password generator built with Python, created as an introduction to cybersecurity concepts and secure password practices.

## Objective

This project was developed to practice basic Python programming and understand how secure passwords can be generated using randomness and different character sets.

## Features

* User-defined password length
* Option to include:

  * Letters (A-Z, a-z)
  * Digits (0-9)
  * Symbols (!@#$...)
* Random password generation

## Technologies

* Python 3
* Built-in libraries:

  * `random`
  * `string`

## How to Run

Clone the repository and run the script:

```bash
git clone https://github.com/yourusername/password-generator.git
cd password-generator
python3 generator.py
```

## Example Usage

```
Type your password length: 12

Choose character types that are going to be in your password:
Letters (y/n): y
Digits (y/n): y
Symbols (y/n): n

Generated password:
aK9dP2xL0qZb
```
